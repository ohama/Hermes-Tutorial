2. "## 개요" — MCP(Model Context Protocol)는 양방향. (a) Hermes를 **클라이언트**로 외부 MCP 서버(GitHub/DB/파일시스템/API)에 연결해 툴을 확장, (b) Hermes를 **서버**로 노출(`hermes mcp serve`)해 Claude Desktop 등 다른 MCP 클라이언트가 사용. ACP는 별개 통합임을 한 줄 예고.

3. "## 개념: ACP와 MCP는 별개입니다" — RESEARCH.md "ACP vs MCP" 표를 제시. ACP = 에디터 네이티브 임베딩(`hermes acp`, VS Code/Zed/JetBrains, `pip install -e '.[acp]'`), MCP = 툴 서버 연결 + Hermes를 MCP 서버로 노출. **이 챕터는 MCP만 다루며 ACP는 간단 언급에 그친다**고 명시(ACP 상세는 본 튜토리얼 범위 밖 — plain prose, 존재하지 않는 챕터 링크 금지).

4. "## 개념: 모드 1 — Hermes를 MCP 클라이언트로 (외부 서버 연결)" — RESEARCH.md "Mode 1":
   - 명령 방식 코드블록(첫 줄 `# 검증: hermes rolling, 2026-06-10`): `hermes mcp` / `hermes mcp catalog` / `hermes mcp install <name>` / `hermes mcp add <name> --preset <type>` / `hermes mcp configure <name>` / `hermes mcp login <server>`.
   - stdio 서버 config YAML(첫 줄 `# 검증: ...`): RESEARCH.md filesystem 예시(command/args/env/tools.include/tools.exclude/timeout/supports_parallel_tool_calls).
   - HTTP 서버 config YAML(첫 줄 `# 검증: ...`): url/headers(Authorization Bearer ${ENV})/ssl_verify.
   - OAuth HTTP 서버 config YAML(첫 줄 `# 검증: ...`): url/auth: oauth, `hermes mcp login <server>`로 PKCE 완료, 토큰 캐시 `~/.hermes/mcp-tokens/<server>.json`.
   - 툴 네이밍: MCP 툴은 `mcp_<server>_<tool>`로 등록(하이픈/점→언더스코어). 재시작 없이 리로드: `/reload-mcp`.

5. "## 실습: 파일시스템 MCP 서버 연결" — 위 stdio 예시를 실습 흐름으로: config.yaml에 mcp_servers.filesystem 추가 → `/reload-mcp` → 에이전트가 `mcp_filesystem_read_file` 등 사용. (예상 출력은 환경 의존이므로 필요 시 placeholder.)

6. "## 개념: 모드 2 — Hermes를 MCP 서버로 노출 (hermes mcp serve)" — RESEARCH.md "Mode 2":
   - 코드블록(첫 줄 `# 검증: hermes rolling, 2026-06-10`): `hermes mcp serve` / `hermes mcp serve --verbose`.
   - **정확성 핵심:** `hermes mcp serve`는 **~10개 플랫폼 메시징 툴**(Telegram/Discord/Slack 등 메시지 관리)을 stdio MCP 서버로 노출하며, 세션 스토어에서 대화 데이터를 읽는다. **전체 "Hermes를 에이전트로 실행"하는 릴레이가 아니다**. 정확한 10개 툴 이름은 미검증이므로:
     ```
     > 검증 필요: `hermes mcp serve`가 노출하는 정확한 툴 이름은

     ```
   - Claude Desktop 연동 JSON 코드블록(주석 줄 `// 검증: hermes rolling, 2026-06-10`): RESEARCH.md `~/.claude/claude_desktop_config.json`의 `mcpServers.hermes` ({command: "hermes", args: ["mcp","serve"]}).
   - Cursor/VS Code/Zed는 동일 `hermes mcp serve` 명령을 쓰지만 설정 파일 경로/UI는 미검증:
     ```

     ```
